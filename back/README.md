# EventHub Backend

Backend for the EventHub application built with Node.js and Express.

## Features
- RESTful API
- User authentication
- Event management

## Installation
1. Clone the repository
2. Run `npm install`
3. Start the server with `npm run dev`

## Docker
1. Build the image: `docker build -t eventhub-backend .`
2. Run the container: `docker run -p 3000:3000 eventhub-backend`
